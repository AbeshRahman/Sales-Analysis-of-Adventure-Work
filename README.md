# Adventure Works Sales Analysis Dashboard

## Project Overview
This project features an interactive **Sales Analysis Dashboard** built in **Microsoft Excel** using the **Adventure Works dataset**. The primary goal was to transform raw transactional data into a dynamic visual tool that delivers actionable business insights into sales performance, customer segments, and regional trends.

The project demonstrates the power of Excelâ€™s advanced data modeling capabilities, moving beyond static spreadsheets to a scalable business intelligence solution.

## Key Features & Insights
The dashboard provides both high-level KPIs and detailed analytical views:

- **Core KPIs**: Real-time tracking of Total Sales, Revenue, Profit, and Quantity Sold  
- **Performance Trends**: Line and column charts showing yearly and monthly sales trends  
- **Product Analysis**: Category-wise comparisons to identify top-performing products  
- **Geographic Insights**: Regional sales distribution across different territories  
- **Customer Segmentation**: Sales insights by customer segments  

## Technical Implementation

### Data Modeling with Power Pivot
- Structured data model using multiple related tables (Sales, Products, Regions, Customers)
- DAX measures for accurate calculations and performance-driven aggregations

### Interactivity & Dynamic Visuals
- **Slicers & Filters** to dynamically analyze data by Year, Region, Product Category, and Customer Segment  
- **Connected Pivot Charts** where selections in one visual update the entire dashboard  
- **Macro-Enabled Workbook (.xlsm)** to support advanced navigation and automated functionality   

## Project Background
This project was completed as a hands-on learning exercise inspired by industry-standard tutorials. It reflects practical experience in data cleaning, data modeling, and dashboard UI/UX design within Microsoft Excel.

## Technologies Used
- Microsoft Excel (Advanced)
- Power Pivot (Data Modeling)
- DAX
- Pivot Tables & Pivot Charts
- Slicers & Timelines
